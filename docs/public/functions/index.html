Functions

Complete API reference for all @sil/color functions, organized by module.

Each module is a focused set of pure functions — import only what you need. The library is fully tree-shakeable with no side effects.

Module Description
Convert Direct conversions between specific formats (e.g. hexToRgb, hslToCmyk)
To Smart helpers that accept any format and convert to a target (e.g. toHex, toRGB)
Manipulate Modify color properties — lighten, darken, mix, set opacity, text contrast
Get Extract individual properties — hue, saturation, lightness, RGB channels
Is Type guards — validate whether a value is a HEX, RGB, HSL, etc.
String Format any color as a CSS-ready string
Brightness Perceived brightness via WCAG luminance
Names Look up human-readable color names
Nearest Find the closest color from a set
Random Generate random, pastel, or bright colors with range constraints
Palette Generate and export cohesive named color palettes
Complementary Generate shades, tints, tones, and hue variants

  • Brightness

    Functions for measuring perceived brightness using WCAG luminance.

  • Complementary

    Generate color scales — shades, tints, tones, or hue rotations — from a base color.

  • Convert

    Direct conversion functions between specific color formats. Each function takes a fixed input format and returns a fixed output format. If you want to convert from any format without knowing the source, use the to helpers instead.

  • Get

    Extract individual properties from any color format.

  • Is (Validation)

    Type-guard functions that check whether a value is a valid color in a specific format. All return boolean.

  • Manipulate

    Functions for modifying color properties. All functions preserve the input color type (HEX in → HEX out, HSL in → HSL out, etc.).

  • Names

    Look up human-readable color names.

  • Nearest

    Find the closest color from a set based on Euclidean RGB distance.

  • Palette

    Generate a cohesive named color palette from one or multiple base colors.

  • Random

    Generate random colors with optional range constraints and output format control.

  • String

    Convert color objects to CSS-ready strings.

  • To

    Smart conversion helpers that accept any color format and convert to a target format. Unlike the direct conversion functions in convert, these handle format detection automatically.
