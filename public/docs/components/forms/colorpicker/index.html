ColorPicker

A color selection component that displays a grid of predefined color options.

Import

import { ColorPicker } from 'hukibs'
import 'hukibs/style.css'

Usage

<ColorPicker 
  v-model="selectedColor" 
  label="Choose a color"
  :colors="['primary', 'secondary', 'accent']"
/>

Live Example

Props

  • block?: string - Block name for BEM classes (default: ‘color-picker’)
  • colors?: string[] - Array of color names to display
  • size?: Size - Component size (small | medium | large)
  • label?: string - Input label
  • inline?: boolean - Display inline
  • disabled?: boolean - Disable interaction
  • description?: string - Helper text

Events

  • update:modelValue - Emitted when color is selected
  • change - Emitted when color is changed
  • touched - Emitted when component is touched
